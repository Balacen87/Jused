## Архитектура браузерной игры **«Судебный процесс»**

Ниже приведена **структура проекта и логическая модель**, ориентированная на простую реализацию (JS / TypeScript). Архитектура рассчитана на **процедурную генерацию дел**, масштабирование и сохранение прогресса.

---

# 1. Общая архитектура

```
Client (Browser)
│
├── UI Layer
│
├── Game Logic
│
├── Case Generator
│
├── Evidence Engine
│
├── Contradiction Analyzer
│
└── Save System
```

# 3. Основные игровые сущности (Models)

## Case (дело)

```javascript
class Case {
    constructor(id, crimeType, defendant) {
        this.id = id
        this.crimeType = crimeType
        this.defendant = defendant
        this.witnesses = []
        this.evidence = []
        this.testimonies = []
        this.truth = null
    }
}
```

---

## Witness (свидетель)

```javascript
class Witness {
    constructor(name, credibility) {
        this.name = name
        this.credibility = credibility
        this.testimonies = []
    }
}
```

Параметры:

```
credibility
bias
relationship
stress
lyingProbability
```

---

## Evidence (доказательство)

```javascript
class Evidence {
    constructor(type, description) {
        this.type = type
        this.description = description
        this.reliability = 0.8
    }
}
```

Типы:

```
physical
video
expert
document
digital
```

---

## Testimony (показания)

```javascript
class Testimony {
    constructor(witnessId, statement) {
        this.witnessId = witnessId
        this.statement = statement
        this.truth = null
    }
}
```

    let crime = randomCrime()

    let defendant = generatePerson()

    let witnesses = generateWitnesses()

# 6. Генерация правды (истины дела)

Игра должна **знать правильный сценарий**.

```
TRUE SCENARIO
│
├ преступник
├ мотив
├ действия
└ улики
```

Пример:

```
Иван украл телефон
в 14:32
в кафе
```

# 8. Система противоречий

Модуль:

```
ContradictionSystem.js
```

---

## Алгоритм

Сравнение фактов.

```
location
time
object
action
appearance
```

---

### Пример

```
Witness A:
"Он был в красной куртке"

Witness B:
"Он был в синей куртке"
```

→ конфликт

---

### Псевдокод

```javascript
function findContradictions(testimonies) {

    contradictions = []

    for each testimonyA
        for each testimonyB

            if factConflict(A, B)
                contradictions.push(A,B)

    return contradictions
}
```
