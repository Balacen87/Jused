Психологическая модель свидетелей (NPC-подобная система)

Чтобы свидетели в игре выглядели живыми и непредсказуемыми, их нужно моделировать не как статические источники фактов, а как агентов со своей психикой.
Каждый свидетель имеет:

характер

мотивацию

уровень страха / давления

стратегию поведения

вероятность и тип лжи

Фактически это упрощённая AI-модель NPC, применяемая к судебным показаниям.

3. Модель характера (Personality)

Лучше всего использовать упрощённую Big Five модель.

type Personality = {
  honesty: number
  courage: number
  anxiety: number
  empathy: number
  impulsivity: number
}

Пример:

{
  "honesty": 0.8,
  "courage": 0.4,
  "anxiety": 0.6,
  "empathy": 0.5,
  "impulsivity": 0.3
}

Влияние:

параметр	эффект
honesty	вероятность говорить правду
courage	сопротивление давлению
anxiety	вероятность путаницы
empathy	защита жертвы
impulsivity	спонтанные ответы

7. Типы стратегий лжи

Если свидетель решает лгать, он выбирает стратегию.

deny
minimize
misidentify
shift_blame
partial_truth
1. Deny
"Я ничего не видел."
2. Minimize
"Он просто стоял рядом."
3. Misidentify
"Это был другой человек."
4. Shift blame
"Я видел, как другой мужчина взял телефон."
5. Partial truth
"Я видел его возле стола."
(но не говорит, что он взял телефон)
8. Алгоритм выбора стратегии
function chooseLieStrategy(witness){

    if(witness.motivation.protectDefendant > 0.6)
        return "deny"

    if(witness.motivation.selfProtection > 0.5)
        return "partial_truth"

    if(witness.personality.impulsivity > 0.6)
        return "misidentify"

    return randomStrategy()
}
9. Алгоритм принятия решения говорить правду

Свидетель взвешивает факторы.

truthScore =
honesty
+ justice
- fear
- protectDefendant
- greed
Псевдокод
function decideTruth(witness){

    score =
      witness.personality.honesty
      + witness.motivation.justice
      - witness.emotionalState.fear
      - witness.motivation.protectDefendant

    if(score > 0.5)
        return "truth"

    return "lie"
}

Пример генерации ответа
function generateAnswer(statement, witness){

    if(witness.emotionalState.stress > 0.7)
        return "Я… не уверен… кажется…"

    if(witness.personality.courage > 0.7)
        return "Я абсолютно уверен."

    return statement
}

17. Итоговый алгоритм поведения
function generateWitnessStatement(witness, trueFact){

    decision = decideTruth(witness)

    if(decision == "truth")
        return generateTrueStatement(trueFact)

    strategy = chooseLieStrategy(witness)

    return generateLie(strategy, trueFact)

}
